# create-pr

Create a pr to the given base branch ( ask if not given )
First gather the current branch name, the commits that will added in pr, then confirm if want to proceed.
Then make sure the current branch is pushed
Make sure to create short concise description and don't add too much jargon ( don't add testing section ) and message automatically, first send to me to review the title, description, then proceed
Then using github mcp, create the pr
